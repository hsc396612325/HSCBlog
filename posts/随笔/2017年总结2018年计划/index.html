2017年总结——2018年计划

2018年1月1日
| 随笔
| 阅读

2017年度总结

2017年这一年匆匆一晃眼，就过去了，总的来说这一年来说过的满意而充实。感觉到了一年盘点收获的时刻，很开心，也挺有成就感的。

2017时间轴

大一下:

  1. 学习c语言下半册,和上学期一样，把所有习题和例题都动手敲了一遍.

  2. 跟着知乎童晶老师做了些c语言小游戏,贪吃蛇，flappy bird, 2048（easyX实现图形化界面）。

  3. 面试进入学校的实验室(西邮移动应用开发实验室),非常开心,认识了一群志同道合的小伙伴们。

  4. 完成c语言课程设计,学生管理系统。

暑假留校:

  1. 开始学习Java。《疯狂java讲义》+视频

  2. 学习Android入门《第一行代码》

  3. 完成第一个小app,计算器

  4. 完成Java的入门

  5. 完成暑假留校任务——聊天室

大二下:

  1. 学习数据结构 《大话数据结构》+课本

  2. 学习数据库(不太扎实)

  3. 继续深入学习Android

  4. 参与实验室多人协作项目《共享书城》《收藏小精灵》

  5. 第一款可以上线的app 《everday》(还没完善好T.T)

2017其他成就

  1. 百词斩累计273天
  2. 跑步累计596.7公里，最远单次15公里
  3. 博客累计34篇
  4. git地址

2018年计划

  1. 寒假完成实验室项目
  2. 寒假刷题准备蓝桥杯
  3. 看完《android艺术开发探索》《java多线程艺术》《android源码和设计模式》《深入了解JVM》
  4. 学好大二下的专业课《计算机网络》《操作系统》
  5. 尝试一次半马
  6. 继续写博客。完成数据结构系列,开始写android方面的博客，以及Java方面的博客。
